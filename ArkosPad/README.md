# ArkosLib

ArkosLib is a simple yet powerfull information management tool.<br>
It has got a topics tree in which you can create hirarchical entries.<br>
<br>
Features:<br>
-Hirarchical topics<br>
-File attachments<br>
-HTML support<br>
-Export to Freemind mindmap (.mm)
